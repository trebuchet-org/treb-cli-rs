## Codebase Patterns
- Sync output uses `println!` for stdout results and `output::print_stage` (eprintln) for stderr progress indicators — don't mix them
- `emoji::CHECK_MARK` is ✓ (U+2713), `emoji::CHECK` is ✅ (U+2705) — sync footer uses CHECK_MARK, tag add/remove uses CHECK
- `color::GREEN` is plain green (Go `FgGreen`), `color::SUCCESS` is green+bold — use GREEN for inline styled text
- Bullet character is `\u{2022}` (•) for Go-matching section bullets
- Golden files for sync/tag human output were updated across review fix commits — they match the Go format
- To disable color in unit tests: call `color::color_enabled(true)` (override_disabled=true), restore with `owo_colors::set_override(true)` after
- Extract `format_*` helper functions that return `String` to make output testable without stdout capture
