## Codebase Patterns
- TestWorkdir auto-creates `.treb/` directory — cannot test "no .treb/" path via golden file framework; use a fixture without `foundry.toml` instead for "uninitialized project" error tests
- EpochNormalizer in `framework/normalizer.rs` replaces epoch timestamps in backup paths (e.g., `prune-<EPOCH>`, `reset-<EPOCH>`, `bak-<EPOCH>`)
- The `regex` crate does NOT support look-behind; use capture groups with alternation instead (e.g., `(prune|reset)-(\d{10,13})`)
- For prune/reset/migrate golden tests, seed registry data via `post_setup_hook` using `Registry::open()` + `insert_deployment()`/`insert_transaction()` directly
- `compose-project` fixture is empty (just `.gitkeep`) — use for tests that need no foundry.toml
