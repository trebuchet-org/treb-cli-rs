# Ralph Progress Log — Phase 06
Started: Tue Mar  4 2026

## Codebase Patterns
- Integration tests use `mod framework;` and `mod helpers;` to import shared test infrastructure
- Golden file tests follow: `TestContext::new("project")` → `IntegrationTest::new(name)` → `run_integration_test(&test, &ctx)`
- `seed_registry(ctx.path())` in a `post_setup_hook` seeds registry with 3 fixture deployments (FPMM, FPMMFactory, TransparentUpgradeableProxy)
- `PathNormalizer::new(vec![ctx.path().display().to_string()])` normalizes temp dir paths to `<PROJECT_ROOT>`
- Setup commands with `treb init` create `.treb/` directory before seed_registry can write deployments
- `UPDATE_GOLDEN=1 cargo test` regenerates golden files from actual output
- `DeploymentStore::list()` sorts by `(created_at, id)` — secondary sort by ID ensures deterministic order when timestamps are equal
