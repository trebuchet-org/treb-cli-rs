## Codebase Patterns
- Integration test shared code goes in `tests/framework/mod.rs` (Rust 2024 edition treats subdirs without main.rs as modules, not test targets)
- Test files that use framework: `mod framework;` at top, then `use framework::workdir::TestWorkdir;`
- `tempfile::TempDir::keep()` replaces deprecated `into_path()` for preserving temp dirs
- Fixture dir helper: `TestWorkdir::fixture_dir("name")` uses `env!("CARGO_MANIFEST_DIR")` for reliable paths
- Symlink items (immutable): lib/, test/, .gitignore, remappings.txt; Copy items (mutable): foundry.toml, treb.toml, src/, script/
