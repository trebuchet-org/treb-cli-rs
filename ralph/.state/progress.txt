## Codebase Patterns
- TransactionStatus variants: Simulated, Queued, Executed, Failed — no "Pending"
- GovernorProposal uses `proposal_id` (not `id`) as the primary key field
- `treb_registry::io::read_json_file` / `write_json_file` are pub in the io module
- RegistryMeta exported from treb_registry crate root; io module is pub
- serde rename_all = "camelCase" on enums changes variant names (BrokenTransactionRef → brokenTransactionRef); use no rename_all to keep PascalCase variant names in JSON
- `toml` crate is in workspace deps but was only a dev-dep in treb-cli; added to main deps for migrate.rs serialization
